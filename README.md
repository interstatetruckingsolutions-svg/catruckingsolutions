# CA Trucking Solutions — Web

Next.js landing page for California IRP, IFTA, and trucking insurance compliance.

## Requirements

- [Node.js](https://nodejs.org/) 20+
- [Git](https://git-scm.com/download/win) (to publish to GitHub)

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Repository

Remote: **[interstatetruckingsolutions-svg/catruckingsolutions](https://github.com/interstatetruckingsolutions-svg/catruckingsolutions)** (`main`).

After local changes:

```bash
git add .
git commit -m "Your message"
git push
```

Use a [Personal Access Token](https://github.com/settings/tokens) for HTTPS if prompted for a password.

## Build

```bash
npm run build
npm start
```
